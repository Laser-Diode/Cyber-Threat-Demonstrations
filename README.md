## Description
This is a repository to store a number of practical outlines of potential cyber attacks that could be carried out on users. These demonstrations are intended to be educational by giving a clear idea of what genuine bad actors could do to a users systems. These guides also outline what can be done to mitigate or minimise the risks of each attack.
